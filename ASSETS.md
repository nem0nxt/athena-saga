# Free 3D Assets for Athena Saga

## Recommended Sources

### 1. Kenney.nl (CC0 License - Completely Free)
- **Character Pack**: https://kenney.nl/assets/character-pack
- **Nature Pack Extended**: https://kenney.nl/assets/nature-pack-extended  
- **Castle Kit**: https://kenney.nl/assets/castle-kit
- **Weapon Pack**: https://kenney.nl/assets/weapon-pack

### 2. OpenGameArt.org
- Search for: "character", "warrior", "medieval"
- Filter by: 3D Models, CC0 or OGA-BY licenses

### 3. Sketchfab (Free Downloads)
- Search: "greek mythology", "warrior", "low poly"
- Filter: Free downloads, CC licenses

### 4. Mixamo (Adobe - Free Characters & Animations)
- Pre-rigged characters
- Free animations (walk, run, attack, idle)
- Export as .fbx for Godot

## Installation Instructions

1. Download assets to `assets/models/`
2. Import in Godot (drag & drop .fbx/.gltf/.obj files)
3. Set up materials in `assets/textures/`
4. Create scenes from models
5. Replace placeholder capsule in main scene

## Asset Format Notes

- **Preferred**: .gltf/.glb (best for Godot)
- **Alternative**: .fbx (good, widely supported)  
- **Basic**: .obj (simple meshes only)
- **Avoid**: .blend (requires Blender import)
