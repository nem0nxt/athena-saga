# Athena Saga

3rd person action-adventure game built with Godot 4.

## Controls

- **WASD** - Movement
- **Mouse** - Camera
- **Space** - Jump
- **Left Click** - Attack
- **ESC** - Toggle mouse capture

## Running

Open with Godot 4.x:
```bash
godot project.godot
```

Or via Flatpak:
```bash
flatpak run org.godotengine.Godot project.godot
```

## Project Structure

```
athena-saga/
├── project.godot     # Main project file
├── scenes/           # Scene files (.tscn)
├── scripts/          # GDScript files (.gd)
└── assets/           # Models, textures, audio
    ├── models/
    ├── textures/
    └── audio/
```
