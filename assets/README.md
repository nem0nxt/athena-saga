# Assets Placeholder

This directory is reserved for future pixel art sprites and audio assets.

## Planned Assets

### Sprites

- `skeleton.png` - Skeleton enemy sprites
- `harpy.png` - Harpy enemy sprites  
- `cyclops.png` - Boss sprite sheet

- `theme.mp3` - Main game music
- `attack.wav` - Spear attack sound
- `hit.wav` - Enemy hit sound
- `block.wav` - Shield block sound
- `powerup.wav` - Item collection sound
- `boss.mp3` - Boss battle music

Currently, all graphics are procedurally rendered using Canvas primitives (colored rectangles and shapes).
